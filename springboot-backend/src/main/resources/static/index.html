企业知识库 RAG 联调页

这个页面由 Spring Boot 静态资源提供，用来快速测试知识库、PDF 入库、会话问答和聊天记录。 页面刷新后会重新从 MySQL 加载知识库、文档和会话。

演示路线

  1. 选择身份：owner、同部门用户、无权限用户。
  2. 创建或选择知识库，上传 PDF，观察重复上传提示。
  3. 创建会话，选择 fastApiDocumentId 后发起 RAG 问答。
  4. 查看聊天记录、引用来源、rerank 分数和权限失败提示。
这部分是面试演示脚本：你可以从上到下点一遍，展示业务后端、权限、文档入库和 RAG 问答闭环。

0. 从数据库恢复状态

刷新页面后，浏览器内存会清空；这里会重新调用后端 GET 接口，从 MySQL 读取已有数据。

1. 创建知识库

这几个按钮用于手动验证权限：owner 和同部门用户可以访问，无权限用户读取文档、读取消息或提问时应该被后端拒绝。
创建成功后，会自动刷新知识库列表，并选中新建的知识库。

2. 上传 PDF

上传成功后，会自动刷新文档列表，并填入 fastApiDocumentId。

3. 创建会话

创建成功后，会自动刷新会话列表，并填入 sessionId。
